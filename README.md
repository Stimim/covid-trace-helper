# covid-trace-helper
A tool to help filling Taiwan covid-19 traces.


# Build
```
python devtools/dev_env.py build
```

# Run local server
```
cd build/
source ../devtools/venv/bin/activate
python main.py
# the local server should be: localhost:8080
```

# Google Cloud API
APIs enabled:
- Cloud Logging API
- Cloud Tasks API
- Cloud Pub/Sub API
- Cloud Datastore API
- Cloud Firestore API
- Cloud Vision API
- Cloud Identity-Aware Proxy API
